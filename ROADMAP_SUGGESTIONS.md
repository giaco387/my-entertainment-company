# 《我的娱乐公司》后续优化建议

## 总体判断

当前游戏已经具备清晰骨架：娱乐公司经营、艺人培养、内容发行、设施建设、随机事件、市场趋势和竞争对手。下一阶段不建议继续单纯堆功能，而是强化三件事：

1. 让玩家知道自己处在哪个经营阶段。
2. 让每个艺人更有记忆点和培养路线。
3. 让设施、内容、艺人状态之间形成更强联动。

核心目标是让玩家随时能判断：我现在最该解决什么问题，我为什么要培养这个艺人，我投入这个设施会改变哪条经营路线。

## 优先级建议

| 优先级 | 方向 | 价值 |
| ------ | ---- | ---- |
| P0 | 新手目标 / 起步任务 | 明确前期目标，减少无从下手 |
| P0 | 总览行动化 | 让玩家每月知道下一步该做什么 |
| P1 | 艺人特质 | 增加养成差异和复玩价值 |
| P1 | 内容发行与设施联动 | 让系统之间互相支撑，减少孤立加成 |
| P2 | 精力正反馈 | 让休息成为策略，而不是纯惩罚 |
| P2 | 分阶段随机事件 | 让事件更像公司成长中的自然结果 |

## 一、前期目标与起步任务

### 现状问题

前期循环是成立的：招人、训练、建直播间、等待出道。但玩家刚开始可能不知道“现在做什么最划算”，容易只是在推进月份。

### 建议方案

增加一组起步任务，显示在公司总览或单独的新手任务区域。

建议任务：

| 任务 | 条件 | 奖励 |
| ---- | ---- | ---- |
| 签约第一位练习生 | 艺人数 >= 1 | 资金 +20 万，知名度 +3 |
| 建造直播间 | 直播间 Lv.1 | 资金 +30 万 |
| 升级训练室 | 训练室 Lv.2 | 知名度 +5 |
| 第一位艺人出道 | 任意艺人状态为已出道/工作中 | 粉丝 +5 万，知名度 +10 |
| 发行第一首单曲 | 任意内容上线 | 资金 +30 万，知名度 +10 |

### 设计原则

- 奖励不宜过大，主要作用是引导。
- 任务应覆盖核心系统，不要强迫玩家走复杂路线。
- 完成后可进入“成长目标”，例如总粉丝 50 万、月净利润为正、拥有 3 名艺人。

## 二、艺人特质系统

### 现状问题

艺人已有唱、跳、演、说唱、粉丝、口碑、精力、稀有度、发展方向，但玩家对单个艺人的策略记忆还可以更强。

### 建议方案

给每位艺人增加 1 个固定特质。特质只做轻量加成，避免变成复杂装备系统。

示例特质：

| 特质 | 效果 | 适合玩法 |
| ---- | ---- | -------- |
| 舞台型 | 工作收入 +10%，工作精力消耗 +10% | 高收益高消耗 |
| 自律型 | 训练成长 +10% | 快速出道 |
| 慢热型 | 训练成长 -5%，出道后粉丝增长 +15% | 长线培养 |
| 综艺感 | 商业演出和签售收益 +15% | 对外业务 |
| 玻璃心 | 口碑低于 50 时精力消耗 +20% | 需要公关保护 |
| 镜头感 | MV / 影像内容收益 +15% | 内容发行 |

### 实现建议

- 在 `artists.js` 的艺人数据中加入 `trait` 字段。
- 在艺人卡片上显示特质标签。
- 初期只实现 5-6 个特质，先验证体验。

## 三、设施阶段感

### 现状判断

设施已经按阶段分组展示，这是正确方向。后续可以把每个阶段做成更明确的成长主题。

建议阶段：

| 阶段 | 设施 | 玩家目标 |
| ---- | ---- | -------- |
| 初创期 | 训练室、录音室、经纪办公室、直播间 | 培养第一位艺人，建立现金流 |
| 培养期 | 员工休息室、服装间、化妆间、艺人护理室、宣传公关部、影像制作室 | 稳定艺人状态，积累形象和曝光 |
| 出道期 | 排练室、商品企划部 | 提升演出收益和粉丝变现 |
| 公司期 | 法务部、经纪人团队、竞争对手 | 处理长期管理和外部竞争 |

### 后续优化

- 每个设施组可以显示“本阶段目标”。
- 设施卡片可以标注“推荐优先级”。
- 建造完成后给出更明确的下一步提示。

## 四、内容发行产品线

### 现状问题

内容类型已经有数字单曲、实体专辑、MV、纪录片，但内容之间和设施之间的联动还可以更强。

### 建议方案

增加内容组合收益，让玩家有发行策略。

示例组合：

| 组合 | 条件 | 效果 |
| ---- | ---- | ---- |
| 单曲 + MV | 同一艺人 2 个月内上线单曲和 MV | MV 收益 +20%，粉丝增长 +10% |
| 专辑 + 纪录片 | 专辑上线前后 2 个月内上线纪录片 | 知名度收益 +30% |
| 直播间 + 单曲 | 直播间 Lv.2 且发行单曲 | 单曲首月收益 +15% |
| 宣传公关部 + 任意发行 | 宣传公关部 Lv.1+ | 发行首月口碑波动风险降低 |
| 影像制作室 + MV | 影像制作室等级越高 | MV 长尾收益提高 |

### 设计原则

- 不要强制组合，单独发行也应可行。
- 组合奖励应鼓励规划，而不是惩罚没规划的玩家。

## 五、精力系统正反馈

### 现状问题

精力目前主要带来风险：低精力降低收益、耗尽后强制休息。这个系统有压力，但正反馈不足。

### 建议方案

加入高精力和主动休息的收益。

可选规则：

| 条件 | 效果 |
| ---- | ---- |
| 精力 >= 80 | 训练成长或工作收益 +10% |
| 连续休息 1 个月 | 下月工作收益 +15% |
| 员工休息室 Lv.2+ | 休息中艺人口碑额外 +1/月 |
| 艺人护理室 Lv.2+ | 精力低于 20 时强制休息前给一次缓冲 |

### 设计目的

让玩家认为“休息是经营选择”，而不是单纯损失一个月收入。

## 六、随机事件分层

### 现状问题

随机事件能增加变化，但如果不分阶段，容易显得只是随机扣数值或加数值。

### 建议方案

按照公司阶段分配事件池。

| 阶段 | 事件方向 |
| ---- | -------- |
| 初创期 | 资金紧张、小范围粉丝增长、练习生受伤、训练突破 |
| 出道期 | 热搜、黑粉、代言邀约、舞台事故、粉丝应援 |
| 成熟期 | 合同纠纷、竞品挖角、平台合作、重大舆论危机 |

### 示例事件

- 初创期：“练习公开视频意外传播” → 粉丝 +3 万，知名度 +5
- 出道期：“舞台直拍出圈” → 粉丝 +15%，口碑 +8
- 成熟期：“竞争对手挖角” → 合同到期艺人续签风险上升

## 七、推荐实施顺序

### 第一阶段：改善前期体验

1. 起步任务系统
2. 总览待办进一步接入任务
3. 设施推荐优先级

### 第二阶段：增强养成深度

1. 艺人特质
2. 特质在训练、工作、内容发行中的加成
3. 艺人卡片信息优化

### 第三阶段：强化系统联动

1. 内容组合收益
2. 精力正反馈
3. 分阶段随机事件

## 八、风险提醒

- 不要一次加入太多新数值，否则玩家难以理解收益来源。
- 艺人特质和发展方向不要功能重复：方向决定长期路线，特质决定个体性格。
- 内容组合不要让普通发行变成“错误选择”。
- 随机事件要避免频繁惩罚玩家，负面事件最好提供补救手段。

## 结论

下一步最值得做的是“起步任务 + 艺人特质”。前者提升新手体验，后者提升长期复玩价值。内容组合和事件分层适合放在第二轮迭代，等基础经营节奏更稳定后再加。
