# 《我的娱乐公司》开发文档 v0.1

## 一、游戏概述

单页面浏览器模拟经营游戏。玩家扮演娱乐公司老板，通过签约艺人、升级设施、参加活动逐步积累资金和声望，最终达成「娱乐帝国」成就。

技术栈：纯 HTML + CSS + Vanilla JS，单文件 524 行，无依赖，无存档。

---

## 二、核心数值

| 资源 | 初始值 | 说明 |
|------|--------|------|
| 资金 | 500 万 | 所有操作的硬通货 |
| 声望 | 12 | 解锁活动、决定公司等级 |
| 月份 | 1 | 手动推进，无上限 |

**公司等级**（仅显示，无额外效果）：

| 声望区间 | 称号 |
|----------|------|
| 0–29 | 初创公司 |
| 30–79 | 新兴公司 |
| 80–199 | 知名公司 |
| 200–499 | 顶级公司 |
| 500+ | 娱乐帝国 |

### 状态与月结算

| 状态 | 每月成本 | 效果 |
|------|----------|------|
| 训练中 | 5 万 | 唱/舞/演随机成长（rap 从不成长）|
| 工作中 / 已出道 | 0 | 产生演出收入，粉丝增长 |
| 休息中 | 2 万 | 无产出 |

演出收入公式：`floor((唱+舞)/12 + random*8) × MV加成`

## 六、已知问题 / Bug

1. **无存档系统** — 刷新页面数据全部丢失
2. **录音室、粉丝见面厅效果未实装** — 建了等于白建
3. **活动无冷却** — 每月可无限重复参加同一活动
4. **说唱值永不成长** — 训练时只有唱/舞/演在涨
5. **声望增长来源单一** — 仅靠 `floor(月收入/25+1)`，活动声望不纳入公式
